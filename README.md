# 要件定義
- `工数目安：2h`
- `ファイル：特になし`

- 非会員
    - ユーザーを新規作成できる。
    - `共通` 全ての求人リストを見ることができる。
    - 指定項目（金額）を見る事ができない。
    - 指定URLへ飛ぶことができない。

- 会員
    - ユーザー情報を更新できる。
    - ユーザー情報を閲覧できる。
    - `共通` 全ての求人リストを見ることができる。
    - 指定項目（金額）を見る事ができる。
    - 指定URLへ飛ぶことができる。
